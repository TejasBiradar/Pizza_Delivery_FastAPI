Pizza Delivery App

This is a pizza delivery application built using FastAPI. The primary purpose of this app is to provide a hands-on experience in understanding the flow and functionalities of the FastAPI framework. It features a range of endpoints for managing pizza orders, including placing, deleting, and tracking orders. Additionally, it serves as a practical exercise to deepen my knowledge of RESTful API design and implementation using FastAPI.
Note: Please test the endpoints in Postman/Insomnia.
